## 🏗️ Built With

- **React** — Single Page Application (Create React App)
- **Tailwind CSS v3** — Custom `neo-*` design tokens
- **Space Grotesk** — Font (weights 700 & 900)
- **Lucide React** — Icons
- **Vercel** — Hosting & auto-deploy on push

---

## 🚀 Running Locally

```bash
git clone https://github.com/MohitMurarka/mohit-portfolio.git
cd mohit-portfolio
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)
